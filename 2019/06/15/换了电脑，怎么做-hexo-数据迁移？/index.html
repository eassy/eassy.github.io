换了电脑，怎么做 hexo 数据迁移？

Jun 15, 2019

使用 hexo 搭建博客的时候，我们的网页数据都存在了本地电脑上，更换电脑时，需要我们做一个同步。我们需要同步的文件并不多：

  1. source 文件夹 含有我们网站的源文件
  2. _config.yml 文件 网站的配置文件
  3. theme 文件夹 主题

接下来就是实际更换流程：

  1. 在新电脑上配置 hexo 环境，按照 hexo 网站教程，搭建环境，并且将 github 支持完成。
  2. 覆盖掉上述文件。

整个替换流程就是这么简单。。
